# BashUpdater

![BashUpdater](bashUpdaterImages/bashupdater3.png)

BashUpdater is a compilation of updating bash scripts

-----------------------
### How to use this?

```$ git clone https://github.com/IUseDebianBtw/BashUpdater.git```

```$ sudo ./updateDebian.sh```
